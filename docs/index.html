Actionly Companion

AI-powered companion app for macOS

Download for macOS

Version 0.1.0

Features

Floating overlay interface
Quick access with keyboard shortcut
AI-powered assistance with Gemini
Screenshot capture and analysis
Runs quietly in the background

Quick Start

1. Download and open the DMG file

2. Drag Actionly Companion to Applications

3. Launch the app (it runs in the background)

4. Press ⌘ + ⇧ + Space to open


Requirements: macOS 13.0 or later
